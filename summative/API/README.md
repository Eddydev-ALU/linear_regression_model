# Life Expectancy Prediction API

FastAPI-based REST API that serves the best-performing regression model (Random Forest / Decision Tree / SGD) trained on the **WHO Life Expectancy** dataset.

## Project Structure

```
life-expectancy-api/
├── main.py               # FastAPI application (prediction + retrain endpoints)
├── predict.py            # Standalone prediction script (Task 1 bridge)
├── requirements.txt      # Python dependencies
├── render.yaml           # Render deployment blueprint
├── best_model.pkl        # Saved best model (generated by Task 1 notebook)
├── scaler.pkl            # Saved StandardScaler
└── feature_names.pkl     # Saved feature column names
```

## Local Setup

```bash
# 1. Install dependencies
pip install -r requirements.txt

# 2. Make sure the .pkl files from Task 1 are in this directory

# 3. Start the server
uvicorn main:app --reload --port 8000

# 4. Open Swagger UI
#    http://127.0.0.1:8000/docs
```

## API Endpoints

| Method | Path       | Description                                        |
|--------|------------|----------------------------------------------------|
| GET    | `/`        | Health check – confirms model is loaded             |
| POST   | `/predict` | Predict life expectancy for a single observation    |
| POST   | `/retrain` | Upload new CSV to retrain all models & save best    |

## Deployment on Render

1. Push this folder to a **GitHub repo**.
2. On [Render](https://render.com), create a **New Web Service** → connect the repo.
3. Render auto-detects `render.yaml`. Confirm settings and deploy.
4. Your public Swagger UI will be at `https://<your-service>.onrender.com/docs`.

## CORS Configuration

The API uses **explicit** CORS origins (not a wildcard `*`):

- `https://life-expectancy-frontend.onrender.com`
- `https://life-expectancy-api.onrender.com`
- `http://localhost:3000` / `5173` / `8080` / `127.0.0.1:8000`

**Allowed methods:** GET, POST, PUT, DELETE  
**Allowed headers:** Content-Type, Authorization, Accept, X-Requested-With  
**Credentials:** Enabled
